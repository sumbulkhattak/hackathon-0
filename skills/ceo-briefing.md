# Skill: CEO Briefing (Monday Morning Briefing)

## Description
Autonomously audit the vault and generate a weekly "Monday Morning CEO Briefing" — a concise executive summary of completed tasks, bottlenecks, activity statistics, and proactive suggestions.

## Trigger
- Scheduled: Every Monday at 07:00 UTC (or on demand)
- Manual: User runs the briefing command

## Output: vault/Briefings/YYYY-MM-DD_Briefing.md
```markdown
---
generated: 2026-02-17T07:00:00Z
period: 2026-02-10 to 2026-02-17
---

# Monday Morning CEO Briefing

## Executive Summary
This period saw 15 total actions: 6 emails processed, 5 plans created, and 0 errors.

## Activity This Period
| Metric | Count |
|--------|-------|
| Emails processed | 6 |
| Plans created | 5 |
| Auto-approved | 2 |
| Manually approved | 3 |
| Rejected | 0 |
| Errors | 0 |

## Completed Tasks
- [x] plan-invoice-client-a.md
- [x] plan-meeting-response.md

## Bottlenecks
| Item | Folder | Waiting |
|------|--------|---------|
| plan-complex-request.md | Pending_Approval | 48 hours |

## Pending Items
- Needs_Action: 3 items
- Pending_Approval: 1 item
- Quarantine: 0 items

## Proactive Suggestions
- Review items in Pending_Approval that have been waiting > 24 hours

---
*Generated by Digital FTE — Monday Morning CEO Briefing*
```

## What It Analyzes
- **Done/** folder: completed items within the reporting period
- **Logs/** folder: action counts (emails sent, plans created, approvals, rejections, errors)
- **Pending_Approval/** and **Needs_Action/**: pending items and bottlenecks (items waiting > 24 hours)
- **Quarantine/**: error items requiring attention

## Implementation
- Module: `src/briefing.py`
  - `generate_briefing(vault_path, period_days=7)` — full briefing markdown
  - `save_briefing(vault_path, content)` — save to Briefings/ folder
  - `get_period_stats(vault_path, period_days)` — log-based activity statistics
  - `get_completed_items(vault_path, period_days)` — recently completed tasks
  - `get_bottlenecks(vault_path)` — items pending > 24 hours
- Tests: `tests/test_briefing.py` (14 tests)
